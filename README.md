# Contact Management app (ConMan)

This is a simple console app that keeps a list of contacts. You can:

1. List all contacts
1. Add contact
1. Search Contact
1. Edit contact
1. Exit

### To run the app

```bash
$ bundle install
$ bin/app
```

### To run the tests

```bash
$ bundle exec rspec --color
```

Or simply [1]:

```bash
$ rspec --color
```

[1] Unless you have

```bash
alias rspec="rspec --color"
```

in your `.bash_profile`.

### Folder structure

* `bin `: Executables
* `lib `: Sources
* `spec`: Tests
